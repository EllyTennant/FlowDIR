TopoToolbox functions

TopoToolbox contains numerous functions that are associated with four main classes:
The class GRIDobj stores a gridded digital elevation model together with its georeferencing.
The class FLOWobj contains local flow directions derived from the GRIDobj. FLOWobj are thus similar to flow direction grids as used in GIS. However, TopoToolbox stores flow directions in a topologically sorted list of linear indices which departs from the gridded representation.
The class STREAMobj contains the geometric representation of the stream network. STREAMobj are derived from FLOWobj and also contain topologically sorted lists of indices that enable efficient computations and modifications of stream networks.
The class SWATHobj combines a set of tools for swath mapping.
Following functions are available for each class
